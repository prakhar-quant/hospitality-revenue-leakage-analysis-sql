# hospitality-revenue-leakage-analysis-sql
SQL-based hospitality revenue leakage and profitability analysis using 134K+ hotel booking records.

## Overview

In this project, I analyzed 134K+ hotel booking records using SQL to understand revenue performance, occupancy trends, booking cancellations, and revenue leakage across multiple hotel properties.

The goal was to identify where revenue is generated, where it is being lost, and provide recommendations to improve profitability.

## Dataset

The project uses:

- fact_bookings
- fact_aggregated_bookings
- dim_hotels
- dim_rooms
- dim_date

## SQL Skills Used

- Joins
- Aggregate Functions
- GROUP BY
- CASE Statements
- Window Functions (RANK)
- KPI Calculations
- Trend Analysis

## Key Insights

- Atliq Exotica generated the highest revenue.
- Atliq Seasons generated the lowest revenue.
- Revenue leakage was mainly caused by booking cancellations.
- Cancellation rate was 24.83%.
- RT2 had the highest number of cancellations.
- Atliq Palace experienced the most cancellations.
- Mumbai generated the highest revenue.
- Delhi achieved the highest occupancy.
- Elite rooms generated the highest revenue.

## Recommendations

- Reduce booking cancellations through stricter policies.
- Promote prepaid and direct bookings.
- Review pricing for RT2 room categories.
- Improve demand for Atliq Seasons through promotions.
- Optimize high-leakage booking channels.

## Project Outcome

This analysis helped identify the major drivers of revenue leakage and provided data-driven recommendations to improve hotel performance and profitability.
